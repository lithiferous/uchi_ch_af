# Uchi.ru ETL test

## Getting Started

Install docker & docker-compose on your local machine. Next up enable docker service (It may be neccessary to reboot).

### Running

To build all containers

```
sudo docker-compose up -d --build
```

To start locally in background

```
sudo docker-compose up -d
```

To start locally verbose

```
sudo docker-compose up
```

Checking compose info

```
sudo docker ps
```
